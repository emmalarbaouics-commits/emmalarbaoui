# Landing page — Diagnostic Financier Personnalisé

Page d'Emma Larbaoui avec le simulateur de patrimoine interactif et la prise de rendez-vous.

## Structure
```
index.html        # page simulateur / diagnostic
assets/           # photos optimisées (WebP + JPEG)
```

Garder le dossier `assets/` à côté de `index.html` (images en chemin relatif). Ouvrir `index.html` dans un navigateur — aucune installation requise.
