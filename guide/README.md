# Landing page — Guide financier gratuit

Page d'Emma Larbaoui pour le téléchargement du guide gratuit et la prise de rendez-vous.

## Structure
```
index.html        # page guide
assets/           # photos optimisées (WebP + JPEG)
```

Garder le dossier `assets/` à côté de `index.html`. Le bouton de téléchargement pointe vers `Guide-Emma-Larbaoui.pdf` : déposer ce PDF à côté de `index.html`.
